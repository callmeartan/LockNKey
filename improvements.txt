

File Encryption/Decryption Feedback:

success or failure of file operations, rather than just printing to the console.

Redundant else Clause in select_file:
The else clause in select_file method sets self.selectedFilePath to None, which is redundant since you're checking for self.
selectedFilePath existence and value in encrypt_decrypt_file. You could simply not set it if no file is selected.
